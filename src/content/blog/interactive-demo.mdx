## How it works

The component above is an [Astro Island](https://docs.astro.build/en/concepts/islands/) — a self-contained interactive widget embedded directly into this MDX file. The JavaScript runs entirely client-side while the rest of the page remains static HTML.

This means we can mix rich prose with interactive math visualizers, simulations, or any other JS-powered tool — all in a single blog post.
